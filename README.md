# spendlog

spendlog is a modern, full-stack web application designed for real-time collaborative household expenditure  tracking.


## Tech Stack

- **Frontend**
    - React
    - TypeScript
    - Tailwind CSS 4
    - Vite
    - Lucide React
    - Recharts
- **Backend**
    - Node.js & Express
    - WebSocket
    - tsx
- **Database**
    - PostgreSQL
- **Infrastructure & Deployment**
    - Docker
